django-projects
---------------
Simple , intermediat to advance django projects to help with different features for develop web applications in Django

| Create virtual env.   under the root of the github repo, activate it and install Django and Pillow 

.. code-block:: shell
  
  python -m venv ./.venv
  source ./.venv/Scripts/activate  # on Window
  source ./.venv/bin/activate   # on Linux 
  
  python -m pip install --upgrade pip
  python -m pip install --upgrade Django
  python -m pip install --upgrade Pillow

| Read first the README.rst for each project 

| We have many Django project and each are categorized into simple, intermidiate and advanced
